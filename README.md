🧼 VB Higienização

Site institucional desenvolvido para apresentar serviços de higienização automotiva, com foco em uma experiência moderna, visual e responsiva.

🔗 Acesse o site: https://vb-higienizacao.vercel.app/

📌 Sobre o projeto

O VB Higienização é um site criado para divulgar serviços de limpeza e higienização de veículos, com uma interface simples e direta, facilitando o contato com clientes e a visualização dos serviços oferecidos.

🚀 Funcionalidades
✅ Página institucional
✅ Exibição de serviços
✅ Seção de imagens (antes/depois ou serviços)
✅ Contato direto com o cliente
✅ Layout responsivo

🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript
Vite
Deploy com Vercel
